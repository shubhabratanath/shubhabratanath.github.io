Micro-services Placement in Fog Computing

Fog computing can perform in-network processing using the residual resources of the network equipment like the routers. As the fog nodes are resource constrained, the fog services need to adopt micro-service architecture. In the micro-service architecture, each application is developed in the form of a collection of loosely coupled lightweight services.

Objective

Place the micro-services in fog devices so that the maximum response time taken by the applications is minimized.
